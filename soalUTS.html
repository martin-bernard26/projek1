Media Pembelajaran Sistem Persamaan Linear

Kemampuan Komunikasi Matematis

Deskripsi Projek:

Buatlah sebuah media pembelajaran interaktif berbasis VBA for Excel yang dapat membantu siswa SMA memahami konsep Sistem Persamaan Linear Dua Variabel (SPLDV). Media yang Anda buat harus mampu memfasilitasi siswa untuk mengkomunikasikan ide-ide matematis mereka baik secara visual, simbolik, maupun verbal.

üéØ Ketentuan Media:

  • Menyediakan input untuk koefisien persamaan linear (ax + by = c)
  • Menampilkan representasi grafik kedua persamaan pada koordinat Kartesius
  • Menunjukkan langkah-langkah penyelesaian dengan metode eliminasi, substitusi, atau grafik
  • Menyediakan fitur penjelasan tekstual untuk setiap langkah penyelesaian
  • Menampilkan interpretasi solusi dalam konteks masalah nyata
  • Menggunakan navigasi yang intuitif dan desain visual yang menarik

Indikator Penilaian Kemampuan Komunikasi Matematis

  • Representasi Visual (Skor: 25)
    Kemampuan media menampilkan grafik persamaan linear dengan akurat, termasuk titik potong, label sumbu, dan grid koordinat yang jelas
  • Representasi Simbolik (Skor: 20)
    Kemampuan media menampilkan notasi matematis yang benar untuk persamaan, variabel, dan solusi menggunakan simbol-simbol matematika standar
  • Ekspresi Verbal/Tekstual (Skor: 25)
    Kemampuan media menyajikan penjelasan langkah-langkah penyelesaian dengan bahasa yang sistematis, logis, dan mudah dipahami
  • Koneksi Antar Representasi (Skor: 15)
    Kemampuan media menghubungkan representasi grafik, aljabar, dan verbal secara koheren sehingga memperkuat pemahaman konsep
  • Interpretasi dan Komunikasi Solusi (Skor: 15)
    Kemampuan media menyampaikan makna solusi dalam konteks masalah, termasuk kasus khusus (tak hingga solusi, tidak ada solusi)

Total Skor: 100

Media Eksplorasi Fungsi dan Transformasinya

Kemampuan Abstraksi Matematis

Deskripsi Projek:

Rancanglah sebuah media pembelajaran interaktif berbasis VBA for Excel yang memungkinkan siswa untuk mengeksplorasi konsep Fungsi dan Transformasinya (fungsi kuadrat, translasi, refleksi, dilatasi). Media ini harus mendorong siswa untuk melakukan abstraksi matematis dari pola-pola yang diamati menuju pemahaman konsep yang lebih umum.

üéØ Ketentuan Media:

  • Menyediakan kontrol interaktif untuk mengubah parameter fungsi (a, b, c pada ax¬≤ + bx + c)
  • Menampilkan grafik fungsi yang berubah secara real-time saat parameter dimodifikasi
  • Menyediakan fitur transformasi (translasi vertikal/horizontal, refleksi, dilatasi)
  • Menampilkan perbandingan grafik fungsi awal dan hasil transformasi
  • Menyediakan fitur untuk mengekstrak pola dan generalisasi dari transformasi
  • Menampilkan tabel nilai yang mendukung identifikasi pola

Indikator Penilaian Kemampuan Abstraksi Matematis

  • Identifikasi Pola (Skor: 20)
    Kemampuan media memfasilitasi pengguna untuk mengidentifikasi pola perubahan grafik berdasarkan perubahan parameter fungsi melalui visualisasi dinamis
  • Generalisasi dari Kasus Khusus (Skor: 25)
    Kemampuan media menunjukkan bagaimana berbagai kasus khusus (contoh: f(x) = x¬≤, f(x) = 2x¬≤, f(x) = -x¬≤) mengarah pada prinsip umum transformasi fungsi
  • Manipulasi Simbolik Abstrak (Skor: 20)
    Kemampuan media menampilkan perubahan bentuk aljabar fungsi ketika transformasi diterapkan (misal: f(x) ‚Üí f(x-h) + k) dan menghubungkannya dengan efek visual
  • Pemisahan Konsep Esensial dari Detail (Skor: 20)
    Kemampuan media membantu pengguna fokus pada konsep inti transformasi tanpa terganggu detail kalkulasi, dengan menyembunyikan/menampilkan informasi sesuai kebutuhan
  • Konstruksi Model Matematis (Skor: 15)
    Kemampuan media memungkinkan pengguna membangun pemahaman abstrak tentang hubungan antara parameter fungsi dan bentuk grafik untuk membuat prediksi

Total Skor: 100
